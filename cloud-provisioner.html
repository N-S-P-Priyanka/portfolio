OBJECTIVE OF OUR PROJECT

Worked on a MVP leveraging LLMs, Python, and IBM Cloud, which streamlined resource management processes. This innovative solution resulted in a 40% reduction in cloud resource management costs, enhancing operational efficiency and providing significant savings for users.

STEPS INVOLVED:

  • User Research and Persona Creation: Conducted in-depth user research to understand the needs of organizations using multi-cloud infrastructure, creating personas to represent key stakeholders, such as IT administrators and cloud architects. These personas guided the feature development by highlighting pain points in managing cloud resources efficiently.
  • Story Creation and MVP Planning: Developed user stories to capture the workflows and objectives of each persona, focusing on streamlining multi-cloud management. Based on these stories, identified the essential features for the Minimum Viable Product (MVP), including automated cloud provisioning, cost optimization, and monitoring tools. Prioritized these functionalities for the MVP to ensure the product met user expectations and solved critical challenges.
  • MVP Development and Completion: Built and refined the MVP, focusing on key features like provisioning resources across multiple clouds using LLMs, automating tasks, and reducing cloud resource management costs. Ensured the MVP was fully functional and underwent testing to ensure usability and performance, completing the project with a reliable solution for multi-cloud management.
